# Titanic Dataset — Exploratory Data Analysis

## Overview
Exploratory Data Analysis on the Titanic dataset to find key factors 
that affected passenger survival.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Findings
1. Female passengers had a 74.2% survival rate vs 18.9% for males
2. 1st class passengers had 63% survival vs 24.2% for 3rd class
3. Best survival combination — 1st class female: 96.8%
4. Worst survival combination — 3rd class male: 13.5%
5. Children had the highest survival rate among age groups: 57.9%

## Dataset
- Source: Titanic dataset
- 891 passengers, 12 columns
- Missing values found in Age (177), Cabin (687), Embarked (2)

## What I Learned
- How to load and explore real datasets using pandas
- How to find missing values and understand data quality
- How to use groupby to find patterns in data
- How to visualize insights using Matplotlib
